Introduction To HTML

HTML files are text files so that you can use any text editor to create your first webpage. HTML is at the core of every webpage, behind every webpage you've ever visited there is HTML code that your browser uses to display elements like text, images and tables. HTML is the standard language for documents designed to be displayed in a web browser. An HTML document include different elements such as text headings, paragraphs, images or tables. These elements are defined using different tags.


Text Paragraph

HTML is based on tags. You can tell the web browser that you want to include a paragraph of text in your webpage by enclosing the text in paragraph tags < p> i.e. < p>I'm a text paragraph< /p>


Opening and Closing Tags

The paragraph element requires both opening and closing tags.
< p> is the opening paragraph tag. It defines the start of the document.
< /p> is the closing tag, which is used to define the end of the element with the forward slash.


HTML Elements

HTML uses tags to define elements. The HTML code example below creates a basic webpage with the three differnet HTML elements:

  1. A Heading- defined using the < h> tag.
  2. A Paragraph- defined using the < p> tag.
  3. An Image- defined with the < img> tag.


Text Heading

HTML includes six levels of headings, which are ranked according to importance. You can use up to 6 level headings in HTML, and the tags for these heading elements are: < h1>, < h2>, < h3>, < h4>, < h5>, < h6>. The different heading levels help to communicate the organization and hierachy of the content on a page.


Container Tags

Many HTML elements require both opening and closing tags, also called container tags. Heading are examples of container tags. Container tags always come with content. The content is what we want the browser to display. Container tags come in pairs and follow this structure opening tag + content + closing tag.


Image Tags

You can add images to your webpage with the < img> tag. Images are not techincally inserted into a webpage, they are linked. You need to tell the browser the path or location of the image file so it can be found and displayed. This path or location is called the source (src) of the image and needs to be included the image tag.

Note: The image tag doesn't require a closing tag. Tag's that do not contain closing tags are called empty tags.


The < head> Tag

Immediately following the opening HTML tag, you will find the head of the document, which is identified by opening and closing head tags. The head of an HTML file contains all of the non-visual elements that help make the page work i.e.
< html>
< head>..................< /head>
< /html>


The < tile> Tag

To place a title on the tab describing the webpage, add a < title> element to your head section i.e.
< html>
< head>
< title>First Webpage< /title>
< /head>
< body>
< p>This is a line of Text < /p>
< /body>
< /html>


Horizontal Lines

To create a horizontal line, use the < hr> tag.


Comments

The browser does not display comments, but they help document the HTML and add descriptions, reminders and other notes, i.e.
< !--Your Comment Goes Here>


The < p> Tag

To create a paragraph, simply type in the < p> element with it's opening and closing tags.


The Single Line Break

Use the < br> to add a single line of text without starting a new paragraph.
Note: The < br> element is an empty HTML element. It has no end tag.


HTML Attributes

Attributes provide additional information about an element or a tag, while also modifying them. Most attributes have a value; the value modifies the attribute.

The Align Attribute

The align attribute is used to specify how the text is aligned


Image Location

You need to put in the image location for the src attribute that is between the quotation marks. Example: if you have a photo named "tree.jpg" in the same folder as the HTML file, your code should like this:
< img src="tree.jpg" alt="">

Note: In case the image cannot be displayed the alt attributes specifies an alternate text that describes the image in words. The alt attribute is required.


Image Resizing

To define the image size, use the width and height attributes. The value can be specified in pixels or percentage:
Example: < img src="tree.jpg" alt="" height="50px" width="50px">


Image Border

By default, an image has no borders. Use the border attribute within the image tag to create a border around the image.
Example: < img src="tree.jpg" alt="" width="50px" height="50px" border="1px">

An Ordered list startes with the < ol> tag, while an unordered list startes with the < ul> tag, and each list item is defined by the < li> tag.
Example:

Creating A Table

Tables are defined using the < table> tag. Tables are divided into table rows with the < tr> tag. Table rows are divided into table columns (table data) with the < td> tag.
Example:
< table>
< tr>
< td>< /td>
< td>< /td>
< /tr>
< /table>


The Border and Colspan Attribute

A border can be added using the border attribute e.g. < table border="2">. A table cell can span two or more columns e.g. < td colspan="2">.

The < a> Tag

Links are also an intergral part of every webpage. You can add links to text or images that will enable the user to click on them in order to be directed to another file on webpage. In HTML, links are defined using the < a> tag. e.g. < a href="">< /a>


The Target Attribute

The target attribute specifies where to open the linked document. Giving a _blank value to your attribute will have the link open in a new window or tab: e.g. target="_blank".


Types Of Element

In HTML, most elements are defined as block level or inline elements.


CSS

CSS stands for Cascading Style Sheets.

  • Cascding refers to the way CSS applies one style on top of another
  • Style Sheets control the look and feel of web documents


Why Use Css

CSS allows you to apply specific styles to specific HTML elements. The main benefit of css is that it allows you to separate style from content. Using just html, all the styles and formatting are in the same place, which becomes rather difficult to maintain as the page grows.


Inline Css

Using an inline style is one of the ways to insert a style sheet. With an inline style, a unique style is applied to a single element. In order to use an inline style, add the style attribute to the relevant tag.


Embedded/Internal Css

Internal styles are defined within the < style> element, inside the head section of an html page.


External Css

With this method, all styling rules are contained in a single text file, which is saved with the .css extension. This css file is then referenced in the html using the < link> tag. The < link> element goes inside the head section.


CSS Syntax

Css is composed of styles rules that the browser interpret and then applies to the corresponding elements in your document. A style rule has three parts: selector, property, and value.


Type Selectors

The most common and easy to understand selectors are type selectors. This selectors target element types on the page.


Id and Class Selectors

Id selectors allow you to style an html attribute that has an id attribute, regardless of their position in the document tree.


Descendant Selectors

These selectors are used to select elements that are descendants of another element. When selceting levels, as you can select as many levels deep as you need to.


Comments

Comments are used to explain your code and may help you when you edit the source code later. Comments are ignored by browsers. A css comments look like this: /*Comment Goes Here*/


Cascade

The final appearance of a webpage is a result of different styling rules. The three main sources of style information that form a cascade are:

  • The stylesheet created by the author of the page.
  • The browser's default styles.
  • Styles specified by the user.


Inheritance

Inheritance refers to the way properties flow through the page. A child element will usually take on the characteristics of the parent elements unless otherwise defined.
For Example:
< html>
< head>
< style>
body {
color: green;
font-family: Arial;
}
< /style>
< /head>
< /html>


The CSS Box Model

All html elements can be considered as boxes. The css box model represents the design and layout of the site. It consists of margins, borders, paddings and actual content. The properties work in the same order: top, right, bottom and left. Every element of the webpage is a box. Css uses the box model to determine how the big boxes are and how to place them


Total Width Of An Element

When working with boxes, it is important to understand how the total width of an element is calculated. For example: The total width of the box with paddings will be the sum of width plus padding left and padding right.


Total Height Of An Element

The total height of an element is calculated the same way as the width. The total element height = Height + top padding + bottom padding + top border + bottom border + top margin + bottom margin.


The Border Property

The css border property allows you to customize th borders of html elements. In order to add a border to the element, you need to specify the size, style and color of the border.


Border Width

The properties of the border cannot be separately. The border-width property specifies the width of the border.


The Border-Style Property

The default value of teh border style is none, which defines no border. There are various styles supported for the border-style property: dotted, dashed, double


CSS Width and Height

To style a < div> elements to have a total width and height of 100px.


The Minimum and Maximum Sizes

To set the minimum and maximum height and width of an elment. You can use the following properties:

  • Min-Width: The minimum width of an element
  • Min-Height: The minimum height of an element
  • Max-Width: The maximum width of an element
  • Max-Height: The maximum height of an element


The Background Color Property

The background color property is used to specify the background color of an element.


The Background Color Values

The color of the background can be defined using the three different formats: a color name. hexadecimal values and RGB.


The Background Image Property

The background image property sets one or several background images in an element. Background image can be set not only for the whole page, but for individual elements as well.


The Background Repeat Property

The background repeat property specifies how background images are not repeated. A background image can be repeated along the horizontal axis, the vertical axis, or are not repeated at all. The repeat-x will repeat as backgorund image only horizontally. The repeat-y will repeat a background image only vertically.


Setting The Value To Inherit

When you set the background repeat property to inherit, it will take the same specified value as the property for the elements parent. For example, we set the body background repeat only horizontally, if we set some paragraph background repeat values as the body elment.


The Background Attachment Property

The background attachment property sets whether a background image is fixed or scrolls with the rest of the page. Even if an element has a scrolling mechanism, a "fixed" background doesn't move with the element.


The Background Attachment Values

You can also set the background attachnment to inherit or scroll. When you set the background attachment to inherit, it will inherit the value from it's parent element. When you set the background attachment to scroll, the background image will scroll with the rest of the content.


The List Style Type Property

The css list properties allows us to set different list item markers. In html, there are two types of lists:

  • Unordered Lists (ul)- the list items are marked with bullets.
  • Ordered Lists (ol)- the list items are marked with numbers or letters.

With css, lists can be styled further, and images can be used as the list item marker. One of the ways is to use the light-style-type property, which can be set to circle, square, dise, lower-alpha.


Setting Styles To Link

Links can be styled with any css property ( e.g., color, font-family, background, etc.). In addition links can be styled differently, depending on what state they are in. The following pseudo selectors are available:
a:link- defines the style for normal unvisited links.
a:visited- defines the style for visited links.
a:active- a link becomes active when you click on it.
a:hover- a link is hovered when the mouse is over it.


Links Text Decoration

By default, text links are underlined by the browser. One of the most commons uses of css with the links to remove the underline.


Setting The Mouse Cursor Style

Css allows you to set your desired cursor style when your mouse over an element. For example, you can change your cursor into a hand icon, help icon, and much more, rather than using th default pointer.


The Cursor Property Values

There are numerous other possible values for the cursor property, such as:

  • Default- default cursor.
  • Crosshair- cursor displays as crosshair.
  • Pointer- cursor displays hand icon.


Display:Block

Every element on a webpage is a rectangular box. The display property determines how that rectangular box behaves. A block element is an element that takes up the fullest width available, with line breaks before and after.


Display:Inline

An inline element only takes up as much width as necessary, and does not force line breaks.


Display:None

Display:none hides an element, so it does not take up any space. The element will be hidden, and the page will be displayed as if the element is not there.


The Visibility Property

The visibility property specifies whether an element is visible or hidden. The most common values are visible and hidden. Hiding an element can be done by setting the display proerty to "none" or the visibility property to "hidden". However notice that these two methods produce different results.


The List Image And Position

There are also other list properties, such as:

  • List-Style-Image- specifies an image to be used as the list item maker.
  • List-Style-Position- specifies the position of the marker box (inside, outside).


The List-Style Property

The list-style property is a short hand property for setting list-style-type, list-style-image and list-style-position. It is used to set all of the list properties in one declaration.


The Table Properties

The look of an html table can be greatly improved with css. The border-collapse property specifies whether the table borders are collpased into a single border or separated as default. If the borders are separates, the border-spacing property can be used to change the spacing.


The Caption-Side Property

The caption-side property specifies the position of a table caption. The values can be set as top or bottom


The Empty-Cells Property

The empty-cells property specifies whether or not to display properties and backgorund on empty cells in a table.
Possible values are:

  1. Show: The borders of an empty cells are rendered.
  2. Hide: The borders of an empty cells are not drawn.


The Table Layout Property

The table layout specifies how the width of a table columns is calculated. The possible values are: auto-when column or cell width are not explicity set, the column width will be in proportion to the amount of content in the cells that make up the column. Fixed-when the column or cell width are not explicity set, the column width will not be affected by the amount of content in the cells that make up the column.
